# fisshing_log
遊びで作った釣りの記録アプリです。
釣れた魚を登録できます。
魚種名、釣れた場所、釣れた時間、メモ、写真のアップロードが可能です。

カレンダー機能には、登録した日付をもとにマークされます。

潮汐機能は、ハリボテです。
